### Architecture Type: **Single Page Application (SPA)**

The application uses a **modular SPA architecture** where:

- `index.html` is the single entry point
- `main.js` acts as the **orchestrator** - loading HTML templates and initializing JavaScript modules
- All navigation happens client-side without full page reloads
- The server returns `index.html` for all routes (SPA fallback)

## ğŸš€ Getting Started

### Prerequisites

- **Node.js** v16 or higher
- **npm** (comes with Node.js)
- **MongoDB Atlas** account (or local MongoDB)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/kawtar123-ess/petit_bristo_final.git
   cd petit_bristo_final
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Configure environment variables**

   Create a `.env` file in the root directory:

   ```env
   MONGODB_URI=mongodb+srv://your-username:your-password@cluster.mongodb.net/petit_bristo
   JWT_SECRET=your-super-secret-key-change-in-production
   PORT=4000
   ```

4. **Start the server**

   ```bash
   npm start
   ```

5. **Open in browser**
   ```
   http://localhost:4000
   ```
